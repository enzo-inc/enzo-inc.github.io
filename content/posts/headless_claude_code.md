New agentic CLIs keep coming out every other week, with tips and tricks baked into them that optimize the context management for the models in the CLI environment. This is different than, say, inside an IDE. For example, prompts, context and tools for Sonnet/Opus are optimized if you're using the Claude Code CLI in a way that is not 1:1 if you were to select Sonnet/Opus as your model of choice in Cursor.
